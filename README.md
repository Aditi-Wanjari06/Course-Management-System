# 📖 Overview

The Course Management System simplifies the management of academic schedules and announcements. It provides a centralized platform where administrators can manage courses and communication while students and teachers can easily access important academic information.

This project demonstrates **full-stack development skills**, including frontend UI design, backend API integration, role-based authentication, and responsive web development.

---

# ✨ Features

✅ Role-based access control (Admin, Teacher, Student)  
✅ Course and schedule management  
✅ Create and manage announcements  
✅ Real-time schedule updates  
✅ Centralized dashboard for academic information  
✅ Responsive design for multiple devices

# ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/course-management-system.git
```

### Navigate to the project folder

```bash
cd course-management-system
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```
